Project Shadow Mapping

3D Graphics

Project Overview

The Shadow Mapping project was a part of Computer Graphics course. The Goals of this project are:

Implementing shadow mapping techniques and integrate them into an interactive graphics demo

Tasks Completed

  • Implement shadow mapping for a light source modeled as a perspective camera.
  • Modify the Texture class to support depth and RGBA8 color textures.
  • Implement and use the FrameBuffer class for rendering depth maps.
  • Utilize a Camera class to generate proper view and transformation matrices.
  • Visualize shadow maps and the light frustum.
  • Integrate shadow mapping techniques into an interactive demo.
  • Create a structured portfolio page showcasing your project.

Reflection

I learned a new concept called gamma correction. When I first noticed that the scene looked too dark, I thought it was simply due to incorrect lighting calculations or improperly passing matrices. However, through this, I realized that the actual cause was the difference between gamma space and linear space. Understanding this distinction helped me fix the lighting issues and create more realistic rendering results.
