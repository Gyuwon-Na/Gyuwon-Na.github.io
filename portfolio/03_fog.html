Project Fog

3D Graphics

Project Overview

Thr Fog project was a part of Computer Graphics course. The Goals of this project are:

Implementing various fog rendering techniques in shaders

Integrating them into an interactive graphics demo

Tasks Completed

  • Modify all vertex shaders to ensure they correctly convert model vertex positions into clip space.
  • Implemented various fog techniques(linear, exponential, and fragcoord_z) in fragment shaders.

Reflection

Although I did not encounter any major issues or errors during this project, I felt it somewhat counterintuitive and unnatural that in linear fog, when the fog near value is greater than the fog far value, objects closer to the camera become invisible while those farther away remain visible. This project significantly deepened my understanding of calculation differences between various fog techniques.
