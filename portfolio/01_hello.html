Project Hello Quad

3D Graphics

Project Overview

The Hello Quad project was a part of Computer Graphics course. The Goals of this project are:

Implementing a setup the foundational rendering features.

Building & running for Windows & Web by displaying a simple animated quad.

Tasks Completed

  • Implemented classes to encapsulate OpenGL Textures, VertexBuffers, IndexBuffers, and VertexArrays
  • Created vertex and frament shaders to display a quad.
  • Updating the Demo logic to feed the animations based on time, mouse position, and mouse wheel.
  • Created my own animations in the shader code to display the quad.

Reflection

I struggled with displaying textures on the window. I wrote code similar to what we did with OpenGL last semester, but the texture didn't show up. It turned out that the issue was with the ordering of binding and std::move. From this experience, I learned how important the correct order of operations is when working with binding buffers.
