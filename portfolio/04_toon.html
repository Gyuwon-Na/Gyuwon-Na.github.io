Project Toon Shading

3D Graphics

Project Overview

The Toon Shading project was a part of Computer Graphics course. The Goals of this project are:

Implementing different toon shading techniques in fragment shaders.

Working with normal transformations using a Normal Matrix.

Implementing Linear Fog within a toon shading pipeline.

Integrating shading techniques into an interactive demo.

Tasks Completed

  • Applied Custom Ranges Toon Shading to allow viewers to define their own lighting thresholds and shading styles.
  • Used Uniform Ranges Toon Shading for evenly spaced shading steps, allowing adjustable levels without manual tuning
  • Used a 1D gradient texture to control diffuse shading levels, allowing for artistic flexibility in light response.
  • Implementing linear fog styles.
  • Smoothed specular highlights with thresholding.
  • Handled both anti-aliased and non-anti-aliased rendering modes for comparison and performance testing.

Reflection

Initially, I used the view vector V by normalizing vPosition and computed the half-vector H as L + V for the specular component. However, this caused the shape of the light source to appear too clearly on the surface, which was not the intended effect. Replacing the eye vector with a constant (0, 0, 1) and computing H as L + Eye helped distribute the specular highlights more evenly across the surface, achieving the desired visual outcome. Additionally, during the implementation of gradient texture toon shading, I initially sampled the texture using nl directly, which resulted in a blended look even without anti-aliasing. This made it clear that precise pixel-centered sampling was necessary to achieve sharp, discrete shading levels.
